### MoviePrac
- Java Servlet과 JSP를 사용하여 사용자 요청에 대한 응답을 생성하고, 데이터베이스와 연동하여 동적인 콘텐츠를 생성하는 방법을 처음 학습하여 구현한 영화 예매 프로젝트 입니다.
<br> 웹 동작의 흐름을 익히게 해준 프로젝트이기도 합니다.
<br>
로그인, 관리자 모드의 영화등록,삭제 <br>
사용자 모드의 영화 목록, 영화 예매, 좌석 선택, 예매내역 확인을 구현하였습니다.<br>
